Changelog

Public release history. Stable channel; signed binaries.

Syntaur ships from github.com/syntaur-systems/syntaur as the stable channel. Each release is signed with Sigstore cosign and accompanied by a .cosign.bundle file for offline verification. Active development moves faster on the upstream repository linked from each release. For the full archive, see all releases on GitHub.
v0.7.44 Stable May 21, 2026

First public stable release on the syntaur-systems channel.

  • Sigstore cosign signing (keyless OIDC) — every binary ships with a .cosign.bundle for offline verification
  • SHA-256 checksum file for fast integrity checks
  • Public security disclosure policy at SECURITY.md
