Privacy

Last updated 2026-05-21

This page is the privacy notice for the Syntaur public marketing site at syntaur.app and the Syntaur Systems GitHub organization. Privacy practices for the Syntaur application itself (which runs on hardware you control) are documented separately in the product's installation materials.

What this site collects

The marketing site at syntaur.app is a static page. It does not load analytics scripts, does not profile you, does not set first-party tracking cookies, and does not embed third-party analytics providers.

The CDN that serves the page (Cloudflare) records standard HTTP request metadata — your IP address, user-agent, requested URL, and timestamp — for the purpose of operating the network and detecting abuse. Cloudflare's own privacy practices apply to that data. We do not access, store, or otherwise use those request logs ourselves.

What we collect when you email us

If you send mail to hello@syntaur.app, partnerships@syntaur.app, support@syntaur.app, security@syntaur.app, or any other @syntaur.app address, that email is routed through Cloudflare Email Routing and forwarded to a mailbox we control. We retain it for as long as we need to act on it. We don't sell, share, or repurpose your message beyond the topic you reached out about.

What the Syntaur application collects

The downloadable Syntaur gateway runs on hardware you choose and stores its state on local disk. It does not, by design, send your personal data to Syntaur Systems. Specific integrations (smart-home vendors, tax filing partners, payment processors) connect to those third parties' services with their own privacy practices; each connection is explicit and per-feature, and you can review and revoke them.

Third parties we explicitly use

  • Cloudflare — CDN, DNS, and email routing for syntaur.app.
  • Resend — outbound transactional email (when we email you from partnerships@syntaur.app etc.).
  • GitHub — code hosting + release distribution.

Contact

Questions about this notice or anything you've sent us: hello@syntaur.app.

This notice will be expanded when the Syntaur application introduces user accounts or any feature that processes personal data on our infrastructure.
